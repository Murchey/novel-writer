# 小说写手多智能体系统

一个基于多智能体协作的小说创作平台，通过四个专业智能体的协同工作，完成从创意到成稿的完整创作流程。

## 功能特性

### 多智能体协作
- **灵感工程师**：创意发散，将用户输入拓展为丰富素材
- **章节内容架构师**：结构设计，将灵感转化为大纲和细纲
- **写手**：内容创作，根据细纲完成章节写作
- **语言润色师**：语言优化，提升内容的文学性和人性化

### 小说类型支持
- **短篇小说**（1-10章）：单一流程完成
- **中篇小说**（11-30章）：分批次规划
- **长篇连载**（30章以上）：分组管理，每10章为一组

### 进度管理
- 自动记录创作进度
- 支持断篇续写
- 长篇连载按10章分组管理

### 风格管理
- 多种预设风格（默认、言情、悬疑、历史、玄奇）
- 支持自定义风格
- 全书风格统一

## 快速开始

### 开始创作
1. 告诉系统你的创意需求（主题、关键词、情感基调等）
2. 系统自动启动灵感工程师
3. 按流程完成创作

### 续写作品
1. 告诉系统要续写的作品
2. 系统读取 `WORKSPACE/CURRENT.MD`
3. 自动完成续写

### 更换风格
1. 指定风格类型（如：使用言情风格）
2. 系统应用对应风格
3. 重新润色相关内容

```
novel-writer/

├── AGENTS/                # 智能体定义

├── SKILLS/                # 技能定义

│   └── Structure_Design.MD     # 结构设计技能
├── STYLES/                # 风格定义

│   ├── Romance_Style.MD  # 言情风格
```

## 使用示例

### 示例1：开始创作新小说
```
用户：我想写一部都市言情小说，主题是"错过与重逢"，情感基调是温暖治愈
系统：启动灵感工程师，进行创意发散...
系统：启动章节内容架构师，设计全书大纲和章节细纲...
系统：启动写手，开始创作第一章...
系统：启动语言润色师，优化语言表达...
```

### 示例2：续写未完成作品
```
用户：请继续写我的小说《都市情缘》
系统：读取 WORKSPACE/CURRENT.MD...
系统：读取最近2章内容...
系统：读取下一章细纲...
系统：基于上下文续写第X章...
```

### 示例3：更换写作风格
```
用户：请用悬疑风格重新润色第3章
系统：读取 STYLES/Mystery_Style.MD...
系统：应用悬疑风格进行润色...
系统：输出润色后的版本...
```

## 工作流程

### 完整创作流程
```
用户输入 → 灵感工程师 → 章节内容架构师 → 写手 → 语言润色师 → 最终输出
```

### 长篇连载流程
```
1-10章创作 → 记录进度 → 11-20章创作 → 记录进度 → ...
```

### 断篇续写流程
```
读取进度 → 读取最近2章 → 读取细纲 → 续写创作
```

## 注意事项

1. **长篇连载**：建议按10章分组管理，每完成10章更新进度记录
2. **续写前**：务必读取最近2章内容，确保情节连贯
3. **风格统一**：全书保持一致的风格，避免风格混乱
4. **人物一致**：注意人物性格、语言的一致性
5. **情节连贯**：保持情节的逻辑性和连贯性
