# 创意发散技能

## 技能描述
创意发散是灵感工程师的核心技能，用于将用户输入拓展为多维度的创意素材。

## 使用场景
当用户提供初步创意需求时，使用此技能进行创意拓展。

## 技能流程
1. **输入分析**：解析用户输入的核心要素
2. **维度拓展**：从多个维度进行创意发散
3. **素材整合**：将发散的创意整理成结构化报告

## 发散维度
### 人物维度
- 主角设定（性格、背景、动机）
- 配角设定（关系、功能、特点）
- 反派设定（动机、手段、威胁）

### 场景维度
- 主要场景（地理、时间、氛围）
- 次要场景（功能、特点）
- 特殊场景（象征意义）

### 情节维度
- 核心冲突（内部/外部）
- 情节转折（关键节点）
- 高潮设计（情感爆发点）

### 主题维度
- 表层主题（故事层面）
- 深层主题（思想层面）
- 象征主题（意象层面）

## 输出模板
```markdown
## 创意发散报告

### 核心概念
[用户输入的核心概念]

### 人物发散
- **主角方向1**：[描述]
- **主角方向2**：[描述]
- **配角方向1**：[描述]

### 场景发散
- **场景方向1**：[描述]
- **场景方向2**：[描述]

### 情节发散
- **冲突方向1**：[描述]
- **冲突方向2**：[描述]

### 主题发散
- **主题方向1**：[描述]
- **主题方向2**：[描述]
```

## 输出路径
- **保存位置**：`WORKSPACE/[书名]/AGENT/Inspirer.md`
- **文件格式**：Markdown
- **覆盖策略**：每次新任务覆盖原文件

## 注意事项
1. 保持开放性，不预设限制
2. 注重创意的独特性
3. 考虑创意的可行性
4. 避免过于常见的套路
